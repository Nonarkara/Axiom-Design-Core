# Logo Assets

Drop Axiom logo files here.

## Expected Files

| File | Use |
|---|---|
| `axiom-wordmark-dark.svg` | Wordmark on light backgrounds (`--paper`, `--panel`) |
| `axiom-wordmark-light.svg` | Wordmark on dark backgrounds |
| `axiom-disc-blue.svg` | Blue disc mark — identity, board headers |
| `axiom-disc-dark.svg` | Dark (ink) disc — masthead / hub use |
| `axiom-primary-dark.svg` | Full lockup (disc + wordmark) on light |
| `axiom-primary-light.svg` | Full lockup on dark |
| `axiom-favicon.png` | 32×32 and 64×64 favicon |
| `axiom-og.png` | Open Graph image 1200×630 |

## Logo Grammar

The mark is built on a φ (1.618) grid:
- Disc diameter to cap-height in φ ratio
- Wordmark length to mark height in φ
- Clear space = one disc-diameter on every side

The one red element in the mark (counter in O or small disc) is the Divine Move — the Mondaine second-hand. It is the only red in the mark. Do not add a second.

## Usage Rules

- Never stretch, rotate, or apply a gradient to the mark
- Never place the mark on a busy or photographic field without a solid background plate
- Never outline the wordmark
- One-color reproduction: ink on paper, or paper on ink
- The red element may become ink in one-color reproduction
- Minimum wordmark width: 88px screen / 22mm print
- Favicon: disc-only, glyph legible at 16px

## Naming Convention

```
axiom-[variant]-[colormode].[ext]
```
