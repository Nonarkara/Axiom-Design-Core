§10.7 Buttons square corners · 700 weight · tabular uppercase
View methodology
Primary = ink fill. Critical = red fill. Ghost = transparent, ink border. Tertiary = no border, no fill, grey text + triangle. No radius. No glow. No icon libraries — solid triangle only (§11.5).

§7 Golden Section — φ = 1.618 primary/secondary split · never 50/50
Primary — 61.8%
Main content, argument, data. The φ proportion makes the split feel natural — not arbitrary, not symmetric. Symmetric halves are static. φ is alive.
Secondary — 38.2%
Provenance, meta, HUD, supporting data. Never a sidebar for decoration.
The Golden Section (1:1.618) for layout splits. Apply to hero/detail, sidebar, card proportion, margin-to-content. Never split content in half unless the function demands symmetry.
§20 Hard Bans if in doubt, it's probably on this list
VISUAL
  • Gradients (except legibility overlay on photo)
  • Drop shadows, glows, blurs, glassmorphism
  • Rounded corners (0–2px max)
  • Pure #000 or pure #fff
  • A second free accent color
  • Font weights 700+ on data
  • Centering dense content
  • Decorative icons, emoji, stock imagery
MOTION / CONTENT / BEHAVIOR
  • Entrance animations, scroll reveals, parallax
  • More than one Divine Move per surface
  • A Move not traced to a function
  • Unicode arrows or chevrons
  • Filler — placeholder copy, dummy sections
  • Data without provenance
  • Pretension, academic plumage, jargon
  • Exclamation marks. Emoji.
